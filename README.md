# ⚠️ Case Study

This project is a study case built while learning NestJS fundamentals.

## Purpose

The goal of this project is to understand the core building blocks 
of NestJS — modules, controllers, services, and dependency injection — 
by building a simple real-world feature from scratch.

## What it does

A REST API that stores and retrieves messages persisted in a plain 
JSON file on disk. No database involved — the focus is on learning 
NestJS architecture, not data storage.

## What I learned

- How NestJS modules, controllers, and services work together
- How dependency injection is implemented in NestJS
- How to handle HTTP requests and responses
- How to read and write data to the filesystem with Node.js

## Study context

This project is part of my transition from Frontend Developer 
(Angular/TypeScript) to Backend & Cloud Developer (NestJS + AWS).
